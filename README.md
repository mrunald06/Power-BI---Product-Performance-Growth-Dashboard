📊 Product Performance & Growth Dashboard

🚀 Overview
An end-to-end Product Analytics dashboard built in Power BI to analyze revenue performance, user behavior, and conversion funnel metrics.
This dashboard simulates an e-commerce product growth environment to drive strategic decision-making.

🎯 Business Objectives
Track product revenue performance
Monitor conversion rate (Sessions → Orders)
Analyze Daily Active Users (DAU)
Identify top-performing cities
Evaluate repeat customer behavior
Compare platform engagement (Android, iOS, Web)

📌 Key KPIs
Revenue: ₹22.31M
Completed Orders: 8,424
Conversion Rate: 16.85%
Daily Active Users: 5,000
